# 📊 Bank Customer Churn Analytics Dashboard

## 📌 Project Overview
The Bank Customer Churn Analytics Dashboard is an interactive Power BI project developed to analyze customer churn and identify the key factors influencing customer retention. The dashboard transforms raw banking data into meaningful visualizations, enabling users to monitor KPIs, explore customer segments, and uncover churn trends through interactive filters and charts.

This project demonstrates skills in data cleaning, data modeling, DAX calculations, and dashboard design to support data-driven business decisions.

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📈 Dashboard Features
- Interactive KPI Cards
- Customer Churn Analysis
- Customer Segmentation
- Country-wise Analysis
- Gender-wise Analysis
- Interactive Slicers & Filters

## 📊 Key Insights
- Identified customer churn patterns.
- Compared churn rates across different countries.
- Analyzed customer distribution by gender and geography.
- Built an interactive dashboard to support business decision-making.
